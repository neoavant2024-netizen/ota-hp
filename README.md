# ホワイト急便 太田 — 公式サイト リニューアル

株式会社ホワイト急便太田（クリーニング業／日本さわやかグループ加盟）の
公式サイト現代風リニューアル。**Astro** による静的サイト（ビルドすると
素のHTML/CSS/JSになるので、どのサーバにもそのまま置けます）。

## 必要なもの
- Node.js（18以上。開発は Node 24 / npm 11 で確認）

## 使い方（Windows）
- **`start.bat`** をダブルクリック → 開発サーバ起動＋ブラウザが開きます（<http://localhost:4321>）
- **`stop.bat`** → 開発サーバを停止
- **`build.bat`** → `dist/` に公開用の静的ファイルを出力。`dist/` の中身をサーバに置けば公開できます

## コマンド（手動）
```
npm install       # 初回のみ
npm run dev       # 開発サーバ
npm run build     # dist/ に本番ビルド
npm run preview   # ビルド結果をローカル確認
```

**`src/data/news.ts` だけを編集すれば完了します。** 他のファイルを触る必要はありません。

  {

    title: '【夏のクリーニングセール】…',
    body: '説明文（省略可）',

## ★ セールのチラシ画像を貼る

広告チラシをそのまま大きく掲載できます。

1. チラシ画像を **`public/images/sale/`** に保存（例: `mofu-futon-sale.jpg`）
2. `src/data/news.ts` の該当セールに `image` を指定するだけ

```ts
{
  date: '2026.07.01',
  category: 'セール',
  title: '【毛布・布団セール】…',
  body: '…',
  image: '/images/sale/mofu-futon-sale.jpg',   // ← チラシのパス
  imageAlt: '毛布・布団セールのご案内チラシ',      // ← 説明（SEO・アクセシビリティ用）
},
```

→ **トップページに「SALE 開催中」の大きな枠でチラシが掲載**され、`/news/` にも表示されます。
クリックすると原寸で拡大表示されます。縦長・横長どちらの比率でもOKです。

> 現在 `public/images/sale/mofu-futon-sale.jpg` は**仮画像**です。実物のチラシを**同じ名前で上書き**してください。

## ★ そのほかの画像について（差し替え前提）

`public/images/` の写真は**すべてフリー素材の仮画像**です（Unsplashライセンス／商用可）。
**御社の写真が用意でき次第、同じファイル名で上書きするだけ**で差し替わります。

| ファイル | 使用箇所 |
|---|---|
| `hero.jpg` | トップのヒーロー1枚目／サービス・会社案内・採用のヒーロー |
| `laundry-fold.jpg` | ヒーロー2枚目／法人カード／ダウン・コート／法人ヒーロー |
| `bedding.jpg` | ヒーロー3枚目／パララックス帯／毛布・布団 |
| `shirts.jpg` | 個人カード／一般クリーニング |
| `leather.jpg` | 皮革・着物・特殊品／プレミアム仕上げのヒーロー |
| `coin.jpg` | コインランドリー／店舗・工場案内のヒーロー |

推奨サイズ：ヒーロー用は横1920px程度、その他は横1400px程度。JPEG。

> **人物写真について**：適切な日本人スタッフのフリー素材が見つからないため、
> 現在は人物が写った写真を使用していません（外国人スタッフの写真は撤去済み）。
> 御社で撮影されたスタッフ・工場の写真をいただければ、採用ページなどに反映します。

## 実装メモ
- デザインシステムは `src/styles/global.css` に集約
- 共通レイアウト（SEO/OGP/JSON-LD/ヘッダー/フッター）は `src/layouts/Base.astro`
- モーション（Lenis滑らかスクロール／GSAPパララックス／リビール／カウントアップ）は `src/scripts/site.js`
- ページ遷移は Astro View Transitions
- 本番ドメインは `astro.config.mjs` の `site`（canonical / sitemap に反映）
- カラー：ベースは紺×白、**ホワイト急便グループのピンク `#F24EA0` をアクセント**に使用
